1. About
________

Cretaceous Runner - a PC game, developed by DromeoLand team on RenderWare game engine.The game

is distributed free. It's prohibited to publish game under different name and modified parts of 

game without developer permission. The product is a self-made and non-profit project, so author

is not libabe for any damage caused by the game.


2. System requirements
_____________________

2.1. Minimal
____________

OS: Windows 2000, XP, Vista, 7, 8, 10

CPU: 1,5 GHz

RAM: 512 MB

Video: 3D with 256 MB RAM

Free disk space: 1 GB


2.2. For HD version
___________________

OS: Windows XP, Vista, 7, 8, 10

CPU: 2 GHz

RAM: 512 MB

Video: 3D with 512 MB RAM minimum

Free disk space: 1 GB


3. Install
__________

Run the Installer and follow the instructions. After install complete, you can run the game

through the strouchs on the desktop and in the main menu, or running gta_sa.exe in the root

folder of the game.


4. Settings and save
____________________

After the first start the game will create the CretaceousRunner 1.2 Files folder in Documents,

which will store settings and save files. They can be moved to other PC's to transfer settings

and game saves.


5. Controls
___________

Mouse - turn camera

W - walk forward

S - walk back

A - left

D - right

C - crouch

LBM - bite

RBM - targeting

Space - run

Left Alt - walk slowly

Left Shift - jump

Left Ctrl - snowboard jump

F - use transport/special attack

Arrows up/down - flight heaght control

Q/E - prevous/next weapon

Esc - exit to menu


6. Language
___________

The game can work in English and Russian languages. The default setting is English. To cahnge the

language, gtart the game and go th Options -> language.


7. Video settings
_________________


Go to the display options to set the desired quality and the screen resolution. Don't place too high

parameters if your video card is too weak. Also install draw distance to maximum. To install HD version,

visit the Readme -> HD -> instructions.


8. Truobleshooting
__________________


8.1. The game doesn't start
___________________________

8.1.1. Remove the game settings. Go to Documents -> CretaceousRunner 1.2 Files and delete the cr_1_2.set file,

which stores display configuration, controls, etc.

8.1.2. RBM on the icon of the game -> Properties, and set the compatibility with Windows XP.

8.1.3. Remove scripts of the game as StreamMemFix.asi and d3d9.dll.



8.2. The game slows down
________________________

8.2.1. Set the screen resolution to lower.

8.2.2. Remove the d3d9.dll file in the root folder, it will turn the graphich acceleration off.

8.2.3. Buy a more powerful PC.


8.3. Mouse doesn't work
_______________________


8.3.1. The engine glitch you can fix in the control settings, or just exiting in the menu.

8.3.2. Try to find some fixing scripts in the Internet.
